# Itinerary Planning Reference

## 1) 行程节奏分级

- **轻松**：每天 2-3 个核心活动，机动时间充足
- **均衡**：每天 3-4 个活动，覆盖与舒适平衡
- **紧凑**：每天 4-5 个活动，依赖严格时间管理

默认使用“均衡”；若有老人/儿童或明确慢游偏好，默认“轻松”。

## 2) 单日编排规则

- 上午安排当天最高优先级活动
- 午间安排就近补给，减少折返
- 下午安排同片区次高优先级活动
- 晚间以轻活动或自由探索为主
- 每天至少预留 1 段机动时间（30-90 分钟）

## 3) 预算估算框架

预算拆成四类：
1. 交通
2. 餐饮
3. 门票/体验
4. 机动开销

输出时给“保守值-舒适值”区间，而非单一金额。

## 4) 可替换策略

为每一天至少准备一组替换：
- 雨天替换（室内优先）
- 排队替换（同主题 nearby 备选）
- 体力替换（低强度版本）

## 5) 日程输出模板

建议每个 Day 使用统一结构：
- 当日主题
- 时间块计划（上午/中午/下午/晚间）
- 交通衔接说明
- 预算区间
- 必做项与可删减项
- 应急替换项

## 6) 质量检查清单

发布前自检：
- 是否存在明显折返路线
- 是否出现连续高强度活动
- 是否缺少用餐与休息时间
- 是否缺少雨天或延误备选
- 是否说明关键假设（营业时间、预约可得性）
